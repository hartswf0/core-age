What This Actually Is

I build measurement infrastructure for generative media production. Not "exploring possibilities" or "probing culture"—I'm instrumenting the triage pipeline where AI output becomes reusable narrative components.

The work: run generative scripts, sort the slop, log the waste ratios, compost what's salvageable into typed blocks with provenance. Most of it's garbage. Some becomes symbolic tilth you can actually plant in.

Neurosymbolic Architecture

The core contribution: building models that reason over structure (ontology graph) and generate new assemblies consistent with typed constraints.

// Word→World Pipeline Input: Natural language prompt ↓ Parse: Extract entities, relations, apparatus cues ↓ Ontology Check: Validate against typed schema ↓ Generate: Compile to scene blocks (LEGOS) ↓ Observe: Log telemetry + failure modes ↓ Transclude: Maintain live links to sources ↓ Output: Executable ritual with provenance
  • Apparatus Grammar: Typed ontology of scene elements (entities, morphisms, observer protocols) from HITCH/CLANCY CLOSE
  • LEGOS Blocks: Atomic narrative components with composition rules—reusable across film/web/workshop
  • Transclusion (Xanadu): Bidirectional links keep sources/claims/scenes connected; edits propagate
  • Observer Protocols: Instrumentation layer measuring handoff completeness, steering variety, boundary thickness
  • Failure Taxonomy: Classify waste: null_choice, uncanny_line_match, withdrawal, structural_drift
  • Provenance Graph: Canon DB traces entity lineage, licensing, role-based permissions across productions
What Gets Measured (Instrumentation Layer)

Handoff Completeness: % of agent transitions (Matrix→Cineosis→CinePrompt) with all required payload fields + downstream consumption verified

Steering Variety Index: Count of distinct feedback variables referenced per liberty claim, normalized by harm space considered

Blueprint→Compile→Trace Rate: Share of prompts mapping to LEGOS blocks, compiling to assets, retaining live source links. Report drop-offs per stage.

Transclusion Freshness: 95th percentile lag between source edit and linked artifact update. Classify: auto/semi/manual.

Boundary Thickness Index: Gradient of similarity vs distance at cultural region edges (Axelrod model) + interaction suppression ratio

Ritual Telemetry Coverage: Schema fill-rate for choice, latency_ms, agency_score fields in Sense Ritual interactions
